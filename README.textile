h1. web-fs

web file system

h1. Installing appengine-jruby

<code>gem install google-appengine</code>

h1. Installing Gems for your App

<pre>
appcfg.rb gem sources --add http://gems.datamapper.org
appcfg.rb gem install sinatra dm-appengine addressable appengine-apis extlib maruku rack syntax json_pure rest-client mime-types
appcfg.rb gem cleanup
</pre>

h1. Running

<pre>
cd web-fs
dev_appserver.rb .
</pre>

h1. Debugging

You can debug the app using irb (jruby irb)

<pre>
cd web-fs
appcfg.rb run -S irb
>> require 'app'
</pre>
